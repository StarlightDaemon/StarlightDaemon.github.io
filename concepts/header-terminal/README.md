## Design Notes

**Theme:** Retro CRT/LCD terminal aesthetic, green phosphor (#00cc00)
**Format:** 125x125px to 150x150px boxes
**Animation:** CSS-only, no JavaScript
**Purpose:** Replace static text in header terminal with visual interest
